这个博客是怎么来的

By Andy Y

2017-12-27 Updated:2017-12-27

Contents

1.在 GitHub 上新建一个空 repo，repo 名称是「你的用户名.github.io」（请将你的用户名替换成github用户名）
2.进入一个安全的目录，执行
npm install -g hexo-cli（安装hexo）
hexo init myBlog（会创建并初始化myBlog目录）
cd myBlog
npm i

3.在myBlog目录中执行
hexo new [博客名]（生成一个”[博客名].md”文件，也就是第一篇博客，可以在这个文件里编辑博客的内容）

4.编辑_config.yml文件
title：[博客主页的标题]
author：[博客的署名]
deploy:
type: git
repo: [第一步创建的空的repo的地址]

5.执行：
npm install hexo-deployer-git –save（安装 git 部署插件）
hexo deploy

6.打开repo的GitHub Pages 功能，然后就可以预览博客了

新建博客，执行：
hexo new [博客名]

更行博客，任何修改之后执行：
hexo generate
hexo deploy
就可以更新博客

改主题：
找一个主题https://github.com/hexojs/hexo/wiki/Themes
点击名字可以进入它的github项目首页
把这个项目clone到myblog/themes目录下面
修改 _config.yml
theme: [上面那个主题的项目名称]
